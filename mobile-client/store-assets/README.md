# Store Listing Assets

This directory contains assets required for app store submissions.

## App Store (iOS) Requirements

### App Icons
- **App Icon**: 1024x1024 PNG (no transparency, no rounded corners)
- **Location**: `icon-1024.png`

### Screenshots
- **iPhone 6.7"**: 1290x2796 or 1284x2778 (iPhone 14 Pro Max, 15 Pro Max)
- **iPhone 6.5"**: 1242x2688 or 1125x2436 (iPhone 11 Pro Max, XS Max)
- **iPhone 5.5"**: 1242x2208 (iPhone 8 Plus, 7 Plus)
- **iPad Pro 12.9"**: 2048x2732 (3rd gen and later)
- **iPad Pro 11"**: 1668x2388 (2nd gen and later)

### App Preview Videos (Optional)
- **Duration**: 15-30 seconds
- **Format**: MOV, MP4, or M4V
- **Resolution**: Same as screenshot requirements

## Google Play Store (Android) Requirements

### App Icons
- **High-res Icon**: 512x512 PNG (32-bit with alpha)
- **Location**: `icon-512.png`

### Feature Graphic
- **Size**: 1024x500 PNG or JPG
- **Location**: `feature-graphic.png`
- **Purpose**: Main promotional image

### Screenshots
- **Phone**: 320dp to 3840dp (width or height)
- **7" Tablet**: 320dp to 3840dp
- **10" Tablet**: 320dp to 3840dp
- **TV**: 1920x1080
- **Wear OS**: 384x384

### Promotional Assets (Optional)
- **Promo Graphic**: 180x120 PNG or JPG
- **TV Banner**: 1280x720 PNG or JPG (for Android TV)

## App Store Descriptions

### Short Description (Google Play)
**Maximum 80 characters**

"View client and cession data on mobile. Offline access, real-time sync."

### Full Description

**Cession Manager Mobile**

Access your client and cession data anywhere with the Cession Manager Mobile app. Designed for field workers and managers who need quick access to client information while away from the office.

**Key Features:**
• View complete client database with contact information
• Track cession payment status and progress
• Search and filter clients by name, CIN, or worker number
• Offline access with automatic data synchronization
• Real-time updates when connected to internet
• Clean, intuitive interface optimized for mobile

**Perfect for:**
• Field representatives visiting clients
• Managers reviewing cession portfolios
• Staff needing quick client lookups
• Teams working in areas with limited connectivity

**Data Security:**
• Read-only access ensures data integrity
• Secure cloud synchronization
• No sensitive data stored permanently on device

**Requirements:**
• Internet connection for initial setup and sync
• Compatible with iOS 13+ and Android 8+

The app automatically syncs with your main Cession Manager system, ensuring you always have access to the most current client and payment information.

cession, client management, field work, offline access, data sync, mobile CRM, payment tracking, client database

### Keywords (Google Play)
cession management, client data, field app, offline sync, payment status, mobile database, CRM mobile, client tracking

## Privacy Policy Requirements

Both app stores require a privacy policy. Key points to address:

1. **Data Collection**: What client data is accessed
2. **Data Usage**: How the data is used within the app
3. **Data Storage**: Local caching and cloud sync
4. **Data Sharing**: No third-party sharing
5. **User Rights**: Data access and deletion requests
6. **Contact Information**: Support email and company details

## Testing Requirements

Before submission:
1. Test on multiple device sizes
2. Verify offline functionality
3. Test data synchronization
4. Check all user flows
5. Validate error handling
6. Test on minimum OS versions
7. Performance testing with large datasets
